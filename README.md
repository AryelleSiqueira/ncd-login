## NCD Login

This is a simple login and register web app for NCD. It is built using [Vue 3](https://v3.vuejs.org/) and [Bootstrap 5](https://getbootstrap.com/docs/5.0/getting-started/introduction/).

### Project setup
```bash
npm install
```
```bash
npm run server
```

### Features
- [x] Login
- [x] Register
- [x] Logout
- [x] Form Validation
